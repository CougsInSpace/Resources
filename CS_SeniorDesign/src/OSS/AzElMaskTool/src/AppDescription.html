Name: CustomApp_AWT_STK_X_GfxAnalysis_AzElMaskTool
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to use the GfxAnalysis Control's AzElMaskTool.
Directions:
  1. Click on the buttons in the right hand panel as they become enabled from top to bottom.
  2. Go to the source code of the application and view the SampleCode.java file if you are interested in how to code each button feature.
  3. Continue until you finally click the Close Scenario button.
